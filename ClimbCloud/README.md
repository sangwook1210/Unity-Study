● 게임 소개<br>

화면을 드래그하여 자동차를 움직여 골대에 최대한 가까이 놓는 게임<br>

● 게임 기획

- GetMouseButtonDown과 GetMouseButtonUp의 좌표의 차를 이용하여 자동차를 움직일 거리를 결정
- 화면의 가운데에 자동차와 골대까지 남은 거리를 표시해주는 UI 표시
- 자동차가 움직일 때마다 소리 출력


● 실행 환경<br><br>
Android, Galaxy S10 5G<br>
